## Начальные команды для установки

```bash
# Создание виртуального окружения
python -m venv venv
# Активация виртуального окружения
venv\Scripts\activate  # для Windows
# Установка зависимостей
pip install django psycopg2-binary django-otp django-two-factor-auth celery redis django-bootstrap4
# Создание проекта
django-admin startproject subscriber_management .
# Создание приложений
python manage.py startapp accounts
python manage.py startapp subscribers
python manage.py startapp logs
```
